# 20260526 BidKing 策划归档 10 风险与源码索引

## 当前实现边界与风险

| 风险 | 当前源码状态 | 建议 |
| --- | --- | --- |
| 文档与源码不一致 | 旧归档说首轮旧阶段已删除，但当前源码仍有三段首轮流程 | 单独开任务决定保留还是再次清理，并更新旧归档 |
| `gameConfig.roles` 残留 3 个未绑定角色 | UI 已过滤，但原 config 仍存在旧角色 | 保留作历史兼容或迁移到过时配置 |
| `Hero.packaged_name` 与原版姓名不同 | 当前运行用三国/古代包装名，原版实名见 20260525 竞买人归档 | 策划定稿时明确“源表行为不变，包装可替换” |
| 1136 行 Item 不宜复制进策划文档 | 本批文件只提炼分类和核心字段，完整行在 generated 表 | 若策划需要逐件校对，另导出 CSV/表格 |
| 系统类 SkillEffect 多为结构化解释 | Category 16-28 已能形成 operation，但不是每类都有完整局内表现 | 逐项按 Sim/商店/战后奖励补表现验收 |
| 公共估值隐藏 | 内部生成估值，但客户端隐藏为 0/0 | 若要做更接近源版的公开估值牌，需要确认 UI 展示口径 |
