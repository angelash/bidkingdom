# 20260526 竞拍之王源码策划归档 08 - AI 与模拟训练

## AI 配置入口

原版 AI 主要由两张表驱动：

| 表 | 行数 | 用途 |
| --- | ---: | --- |
| `RankMap` | 83 | 每个 BidMap 的匹配时间、角色权重、最低出价范围、出价粒度 |
| `RankAi` | 120 | 每个角色每回合的出价倍率、用道具概率、出价时间和对抗出价参数 |

## RankMap 字段

| 字段 | 含义 |
| --- | --- |
| `id` | 对应 `BidMap.id` |
| `match_time` | Bot/匹配等待时间权重，如 `[11,15,1000]` |
| `role_spawn` | 20 个角色的出现权重 |
| `min_bid_range` | 最低起拍价范围权重 |
| `bid_type` | 出价粒度和权重 |

例子：`2101` 快递地图中，`role_spawn` 会提高索菲、玛丽亚、维克托等角色权重；医院快递会显著提高海琳娜权重；古董街快递会提高法蒂玛和吴起灵权重。这说明 Bot 不是完全随机，地图类型会影响对手角色分布。

## RankAi 字段

| 字段 | 含义 |
| --- | --- |
| `id` | 角色 + 回合组合 id，例如 1011 表示角色 101 第 1 回合 |
| `role_id` | 对应 `Hero.id` |
| `round_count` | 第几回合 |
| `min_bid_ratio` | 基于估值的最低出价倍率权重 |
| `item_use_probability` | 该回合使用道具概率，千分制倾向 |
| `item_usage_group` | 可用道具组和权重 |
| `bid_time` | 出价时间分布 |
| `bid_pk` | 对抗加价/压价参数 |

通用趋势：

- 第 1 回合道具概率常见 700。
- 第 2 回合下降到 600。
- 第 3 回合下降到 500。
- 第 4 回合下降到 400。
- 第 5 回合下降到 300。
- 第 6 回合通常 0，并清空道具组。

这表示 Bot 会更倾向前中期用道具获取信息，后期进入竞价收口。

## 模拟训练

`Sim` 表 101 行，配合 `PlayerManager` 的 Sim 系列协议形成单人训练/爬层玩法：

| 协议 | 用途 |
| --- | --- |
| `C2S_124_create_sim_game` | 创建训练局，传入 `HeroCid` |
| `C2S_126_sim_game_bid_price` | 训练局出价 |
| `C2S_128_sim_game_use_item` | 使用训练局选择道具 |
| `C2S_156_sim_game_use_buff_item` | 使用训练局 Buff 道具 |
| `C2S_130_get_sim_game_log` | 获取训练记录 |
| `C2S_132_sim_game_shop_buy_item` | 训练商店购买道具 |
| `C2S_134_sim_game_select_win_game` | 选择训练胜利奖励 |
| `C2S_136_sim_game_stand_down` | 训练局弃权 |

训练模式在 `Battle_Handler` 中是 `BattleType.Training`。它与普通局共用 `Battle_Main` 展示，但道具列表来自：

- `SimSelectItemList`：训练局已选择道具。
- `SimBuffItemList`：训练局 Buff 道具，可带条件或目标格。

## 训练局道具池

`Drop 801` 是“个人模拟测试”奖励池，包含大量 `Item_Type=8` 的模拟宝藏道具，例如：

- 放大镜。
- 家居目录。
- 药瓶标签。
- 时尚杂志。
- 军刀鞘。
- 矿石/古董/数码/交通/食饮/书画相关提示物。

训练局的核心不是复用玩家真实背包，而是构造一套独立的模拟资源、商店和奖励。

## LevelUp 与通行证

`LevelUp` 表 256 行，既有收藏等级奖励，也有 Bass/BigBass 通行证奖励。前 40 级可见规律：

- 收藏等级奖励穿插金币、道具箱、门票、局内道具、礼盒。
- Bass 奖励和 BigBass 奖励是独立列。
- 经验/收藏值逐级递增。

这说明模拟、通行证、长期收藏成长都围绕竞拍核心循环发放资源。

## AI 设计结论

原版 Bot 的设计不是“随机填人”，而是表驱动：

- 地图决定角色池、最低出价范围和出价粒度。
- 角色 + 回合决定用道具概率、出价时间和竞价风险。
- 训练模式复用局内展示，但用独立协议、独立道具和独立奖励。

对复刻而言，`RankMap` 和 `RankAi` 是还原“像真实对手”的关键配置，不是可忽略的边角表。
