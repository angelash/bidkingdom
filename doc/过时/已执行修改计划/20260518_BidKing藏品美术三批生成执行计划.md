# BidKing 藏品美术三批生成执行计划

整理时间：2026-05-18

## 目标

基于 `reverse/bidking/generated_prompts/bidking_sanguo_item_prompts.json` 的 590 条藏品 prompt，分 3 个后续对话生成《珍宝局》三国/古代包装透明 PNG 资产。每个资产必须保留原表 `itemId`、`quality`、`baseValue`、`slotType`、`footprint` 和输出路径，不复用原始商业图片、模型或文本。

## 必读参考文件

- `bitkingdom/reverse/bidking/generated_prompts/bidking_sanguo_item_prompts.json`：590 条源 prompt、目标画布、输出路径。
- `bitkingdom/doc/20260518_BidKing藏品与技能原表复刻校准报告.md`：藏品字段口径、无赝品/修复费、`slot_type` 画布规则。
- `bitkingdom/doc/20260518_BidKing配置逻辑等价复刻实施计划.md`：只保留等价结构和三国包装，不提交商业资源。
- `bitkingdom/tools/bidking-item-art.config.json`：批次定义、样本、输出目录、自检阈值。
- `bitkingdom/tools/normalize-bidking-item-art.py`：把生成图去绿幕并规整到目标透明 PNG。
- `bitkingdom/tools/validate-bidking-item-art.py`：检查尺寸、透明度、裁切、绿幕残留。

## 批次设计

| 批次 | `sourceTypeIds` | 数量 | 主题 |
| --- | --- | ---: | --- |
| 1 | `100,101,102,103,109` | 228 | 秘藏、府邸日用、医药军需、锦衣冠服、酒食香料 |
| 2 | `104,105,106` | 179 | 军械兵装、矿玉珠宝、文玩古器 |
| 3 | `107,108,110` | 183 | 机巧器具、车马舟楫、书画典籍 |

这样分组的原因：

- 每批在题材上相对一致，便于判断风格是否跑偏。
- 每批都覆盖多种品质和 `slotType`，不会只验证小方图。
- 第 3 批单独处理 `数码电子 -> 机巧器具`、`交通工具 -> 车马舟楫` 这类更容易语义跑偏的转换。

## 单件生成步骤

每件藏品按以下顺序执行：

1. 从 `bidking_sanguo_item_prompts.json` 读取该条完整 prompt。
2. 使用内置 `image_gen` 生成一张纯 `#00ff00` 绿幕背景源图。
3. 从 `$CODEX_HOME/generated_images/...` 复制最新生成图到：
   - `bitkingdom/tmp/bidking-item-art/raw/bidking_item_<itemId>_raw.png`
4. 运行规整工具：

```bash
python tools/normalize-bidking-item-art.py --item-id <itemId> --input tmp/bidking-item-art/raw/bidking_item_<itemId>_raw.png
```

5. 运行单件自检：

```bash
python tools/validate-bidking-item-art.py --item-id <itemId>
```

6. 若失败，按错误重生成一次。优先修正：
   - 绿幕不纯：强调 “perfectly flat solid #00ff00 background only”。
   - 裁切：强调 “generous 8-12% padding”。
   - 形状不对：强调 `footprint` 比例，例如 `5:2 horizontal silhouette`。
   - 题材跑偏：强调 `keep the gameplay identity as <sourceType>`。

## 每批验收步骤

每批完成后：

1. 运行该批所有 item 的自检，不能有失败项。
2. 生成一张或多张联排预览图，放到：
   - `bitkingdom/tmp/bidking-item-art/review/`
3. 人工检查：
   - 物体是否是单体资产，而不是场景图。
   - 是否无文字、无 UI 框、无标签、无水印。
   - 是否符合三国/古代珍宝包装。
   - 横竖和宽高是否与 `footprint` 视觉一致。
   - 机巧器具不能变成现代电子产品，车马舟楫不能变成现代汽车。
4. 最终 PNG 只落到：
   - `bitkingdom/apps/web/public/art/generated/bidking/items/bidking_item_<itemId>.png`
