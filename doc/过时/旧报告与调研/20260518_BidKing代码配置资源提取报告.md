### 关键代码入口

- 表读取：`code/decompiled/Scripts/AssetLoader.cs`
  - `LoadTable()` 从 `FirstTools.basePath` 读取 `Tables/*.txt`，再调用 `Base64ToTxt.DecodeBase64ToTxt()`。
- 表解码：`code/decompiled/NotHotUpdate/Base64ToTxt.cs`
  - 对整个文本文件做 `Convert.FromBase64String()`，再按 UTF-8 文本使用。
- 热更入口：`code/decompiled/NotHotUpdate/GameInit.cs`
  - `StartApp()` 调用 `FirstTools.LoadAssembly()`，再 `FirstTools.LoadGameLogic()`。
- 程序集还原与加载：`code/decompiled/NotHotUpdate/FirstTools.cs`
  - AOT metadata 和 `Scripts.dll` 都通过 `CryptoUtils.DecryptRaw(..., "ryrs")` 还原。
- XOR 实现：`code/decompiled/NotHotUpdate/CryptoUtils.cs`
  - 简单循环 XOR。
- 资源加载：`code/decompiled/Scripts/AssetLoader.cs`
  - Bundle 名规则：`path.ToLower() + ".data"`
  - Asset 路径规则：`Assets/Game/Bundle/<path><ext>`
- 竞拍格子主板：`code/decompiled/Scripts/Auction.cs`
- 战斗/竞拍主 UI：`code/decompiled/Scripts/Battle_Main.cs`
- 随机竞拍入口：`code/decompiled/Scripts/BattleRandom_Main.cs`

## 资源提取

### 可用工具调研

可用工具：

- AssetRipper：适合整项目/Unity 原生格式导出，支持 Unity serialized files 与 AssetBundle。
- AssetStudio：适合 GUI 浏览和导出，支持 Texture2D/Sprite/AudioClip/Mesh/MonoBehaviour 等类型；原仓库已归档但仍常用。
- UnityPy：适合脚本化、可审计、按 Bundle/类型/名称筛选导出。本项目已采用 UnityPy。

本地新增导出器：

- `bitkingdom/tools/export_unity_assetbundle_assets.py`

支持类型：

- `Texture2D` / `Sprite` -> PNG
- `AudioClip` -> WAV
- `Mesh` -> OBJ
- 可选：`TextAsset`、`Shader`、`Font`、`MonoBehaviour`、`GameObject`

安全默认：

- 默认建议先 `--dry-run` 生成候选清单。
- 真正写出资源文件需要显式增加 `--confirm-private-analysis`。
- 支持 `--bundle`、`--bundle-glob`、`--include-types`、`--name-regex`、`--max-bundles`、`--max-objects` 做窄范围导出。

已验证 dry-run：

- `model/items/*.data`：可识别 `Mesh`、`Texture2D`。
- `ui/prefab/battle/battle.data`：可识别 `Sprite`、`Texture2D`。
- `sound/*.data`：可识别 `AudioClip`。

## 对复刻计划的直接价值

1. “同仓多轮竞拍”核心配置可从 `BidMap + RankMap + Drop + Item + Skill + Hero` 组合复原。
2. `BidMap.auction_rounds_rate` 与当前计划中的成交线高度吻合，可以作为复刻比例来源。
3. `Hero` 和 `Skill` 已经包含大量“信息揭示型能力”，可直接映射到珍宝局的角色技能系统。
4. `Item.slot_type` 是物品占格形状，`item_quality` 是品质，`base_value` 是估值基础。
5. 资源包名和代码路径规则明确，可以用 `icon_path + icon_atlas + model_3D` 建立素材引用表。
6. `battle.data`、`battleintelligence*.data`、`battlerandom.data`、`battleprevpanel.data` 是 UI/流程复刻的优先参考对象。
