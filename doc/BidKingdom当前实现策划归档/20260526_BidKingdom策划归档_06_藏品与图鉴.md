## 藏品字段解释

| 字段 | 策划意义 |
| --- | --- |
| `item_quality` | 品质 1-6，对应普通到典藏 |
| `base_value` | 真实价值，也是结算/图鉴经验的基础 |
| `slot_type` | 占格尺寸，十位为宽、个位为高，例如 35 表示 3x5 |
| `item_type_id / item_type_ids` | 主类型和多类型标签，技能命中依赖它 |
| `collection` | 图鉴/藏品集合 id |
| `collection_coin` | 收藏柜每秒收益基础，前端显示为每小时收益 |
| `is_tradable / is_sale / binds_on_purchase` | 交易、出售、绑定开关 |
| `max_stack_size / max_per_listing` | 市场上架堆叠和单笔上限 |
| `transaction_tax_rate` | 交易税率阶梯 |
| `auction_baseprice` | 交易/拍卖基础价格字段 |
| `skills` | 角色道具、体验卡、部分物品的关联技能或 Hero |
| `drop_group_id` | 可作为掉落组反查 |
