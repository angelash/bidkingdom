# 20260526 BidKingdom 策划归档 07 门槛解锁与限制

## 入场限制

`bidKingBidMapAccess` 综合以下条件：

| 限制 | 来源 | 解释 |
| --- | --- | --- |
| 拍场存在且可见 | `BidMap.is_visiable` | 不可见则不可进 |
| 大场景开放 | `Map.is_open` | 未开放则不可进 |
| 余额门槛 | `BidMap.required_items` | 主要看银币余额 |
| 入场费 | `BidMap.currency_cost`、`item_cost` | 可扣金币/物品凭证 |
| 每日次数 | `Map.daily_counts` | 记录在 profile.dailyMapEntries |
| 开放时间 | `Map.open_time` | 按本地时间窗口判断 |
| 世界进度 | `Map.world_process` + `Constant.world_process` | 由最高成交总值等档案统计解锁 |
| 凭证物品 | `item_cost` | 库存不足则不可进 |
