## 状态口径

- `Equivalent`：核心字段、触发时机和可见性与源码协议等价，允许本项目包装文案不同。
- `Socket substitute`：行为由 Socket.IO 事件和 `matchSnapshot/eventLog/toast` 替代，未使用原 Protobuf 消息号。
- `REST substitute`：行为由 HTTP API 和 profile snapshot 替代，未使用原 Protobuf 消息号。
- `Partial`：存在同域状态或部分行为，但字段、时机、错误响应或状态机不完整。
- `Missing`：当前没有可执行入口或稳定状态承载。
- `Out of scope`：测试/GM/源码调试协议，当前用户可玩流程不要求实现。
