# 本地固定端口与服务脚本

## 固定地址

- Web：`http://127.0.0.1:5188`
- 管理后台：`http://127.0.0.1:5188/admin`
- API / Socket 服务：`http://127.0.0.1:8787`

## 脚本

在 `bitkingdom` 目录执行：

```powershell
npm run dev
```

前台启动 Web 与服务端，固定使用 `5188` 和 `8787`。端口被占用时不会自动换端口，会直接报错。

公网映射调试使用：

```powershell
npm run dev:public
```

该脚本匹配当前映射：

- Web：本机 `5188`，公网 `https://kingdom-web.iepose.cn`
- API / Socket：本机 `5187`，公网 `https://kingdom-server.iepose.cn`

前端会通过 `VITE_SERVER_URL=https://kingdom-server.iepose.cn` 请求注册、登录和 Socket.IO 服务。

```powershell
npm run dev:start
npm run dev:stop
npm run dev:restart
```

后台启动、停止、重启固定端口服务。`dev:stop` 会同时清理旧开发端口 `5177`、`8798` 上残留的旧进程，避免访问到过期实例。

公网映射后台脚本：

```powershell
npm run dev:public:start
npm run dev:public:stop
npm run dev:public:restart
```
